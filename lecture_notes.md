# 1. Правила електробезпеки при роботі з внутрішніми компонентами ПК

![Ремонт комп'ютера](https://upload.wikimedia.org/wikipedia/commons/6/6b/Computer_repair.jpg)

Під час роботи з внутрішніми компонентами комп’ютера потрібно дотримуватися правил електробезпеки. Це допомагає уникнути ураження електричним струмом і пошкодження обладнання.

## Основні правила

- Перед відкриттям корпусу **вимкнути комп’ютер**.
- **Від’єднати кабель живлення** від електромережі.
- Працювати тільки **сухими руками**.
- Бажано використовувати **антистатичний браслет**.
- Не торкатися контактів мікросхем.
- Встановлювати комплектуючі **обережно без сильного натискання**.
- Перед роботою **розрядити статичну електрику**, доторкнувшись до металевої частини корпусу.

![Антистатичний браслет](https://upload.wikimedia.org/wikipedia/commons/3/3b/Antistatic_wrist_strap.jpg)

# 2. Форм-фактори корпусів настільних комп'ютерів

Форм-фактор корпусу визначає **розміри корпусу та сумісність з материнською платою і компонентами**.

![Комп'ютерні корпуси](https://upload.wikimedia.org/wikipedia/commons/2/2e/Computer_cases.jpg)

## Основні типи корпусів

### Full Tower
- великий корпус;
- використовується для потужних комп’ютерів;
- можна встановити багато комплектуючих.

![Full Tower](https://upload.wikimedia.org/wikipedia/commons/9/9a/Computer_case_full_tower.jpg)

### Mid Tower
- найпоширеніший тип корпусу;
- підходить для більшості домашніх ПК.

### Mini Tower
- компактний корпус;
- часто використовується для офісних комп’ютерів.

### Desktop
- горизонтальний корпус;
- зазвичай використовується в офісах.

![Desktop корпус](https://upload.wikimedia.org/wikipedia/commons/5/5c/Desktop_computer_case.jpg)

# 3. Блок живлення комп’ютера

![Блок живлення](https://upload.wikimedia.org/wikipedia/commons/3/3c/ATX_power_supply.jpg)

**Блок живлення (Power Supply Unit, PSU)** — це пристрій, який перетворює електроенергію з мережі у напругу, необхідну для роботи всіх компонентів комп’ютера.

## Основні характеристики

- **Потужність (Вт)** — максимальна кількість енергії.
- **Коефіцієнт корисної дії (ККД)** — ефективність роботи.
- **Система охолодження** — вентилятор для відведення тепла.
- **Система захисту**:
  - від короткого замикання
  - від перевантаження
  - від перегріву

## Основні конектори блоку живлення

![Конектори живлення](https://upload.wikimedia.org/wikipedia/commons/8/8f/Computer_power_connectors.jpg)

- **24-pin ATX** — живлення материнської плати  
- **4 або 8 pin CPU** — живлення процесора  
- **6+2 pin PCI-E** — живлення відеокарти  
- **SATA** — живлення SSD та HDD  
- **Molex** — живлення старих пристроїв

# 4. Основні компоненти материнської плати

![Материнська плата](https://upload.wikimedia.org/wikipedia/commons/3/3a/Computer-motherboard.jpg)

Материнська плата — це головна плата комп’ютера, яка з’єднує всі компоненти системи.

## Основні компоненти материнської плати (7)

### 1. Сокет процесора
Місце встановлення центрального процесора.

### 2. Чіпсет
Керує взаємодією між процесором, пам’яттю та іншими пристроями.

### 3. Слоти оперативної пам’яті (RAM)

![Слоти RAM](https://upload.wikimedia.org/wikipedia/commons/0/0f/DDR4_DIMM_slots.jpg)

Призначені для встановлення модулів оперативної пам’яті.

### 4. Слоти розширення PCI Express
Використовуються для відеокарт та інших плат розширення.

### 5. BIOS / UEFI

![BIOS чіп](https://upload.wikimedia.org/wikipedia/commons/5/5f/BIOS_chip.jpg)

Мікросхема, яка містить програму запуску комп’ютера.

### 6. Роз’єми живлення
Забезпечують подачу електроенергії на материнську плату.

### 7. Порти введення/виведення (I/O)

![Порти материнської плати](https://upload.wikimedia.org/wikipedia/commons/6/67/Motherboard_back_panel_ports.jpg)

USB, HDMI, LAN, аудіо та інші роз’єми.
